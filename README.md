# AI-Fit

AI destekli kişisel kıyafet öneri platformu. Beden ölçülerin, fiziksel özelliklerin ve stil tercihlerine göre sana en uygun ürünleri önerir, AR (Artırılmış Gerçeklik) ile sanal deneme yapmana olanak sağlar.

## Teknolojiler

- Vue 3 + TypeScript + Vite
- Vuetify 3
- Pinia (State Management)
- Vue Router
- Vue Sonner (Toast Notifications)
- Express.js (Backend)
- Replicate AI (AR Try-On)
